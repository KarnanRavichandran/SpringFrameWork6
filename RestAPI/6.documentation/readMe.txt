Add dependency,
	<dependency>
		<groupId>org.springdoc</groupId>
		<artifactId>springdoc-openapi-starter-webmvc-ui</artifactId>
		<version>2.0.0</version>
	</dependency>

Go to http://localhost:8080/swagger-ui/index.html to see the auto documentation of OpenApi

- Remove Dependency later to avoid errors.
